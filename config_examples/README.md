# 配置示例

本目录包含各种服务的配置示例，请根据你使用的服务选择相应的配置。

## 📋 **配置文件说明**

### 🔧 **评价器配置** (`.env.local.evaluator`)

评价器负责对RAG系统的回答进行评分，需要配置Chat模型和Embedding模型。

### 🔌 **RAG系统配置** (`.env.local.*`)

配置要评价的RAG系统，支持多个系统同时评价。

## 🚀 **快速配置指南**

1. 复制对应的示例文件到项目根目录
2. 重命名为 `.env.local.*` 格式
3. 填入你的API密钥
4. 运行评价程序

## ⚠️ **重要提醒**

- 所有 `.env.local.*` 文件都包含敏感信息，不要提交到Git
- 确保API密钥的安全性
- 测试配置是否正确后再运行完整评价
